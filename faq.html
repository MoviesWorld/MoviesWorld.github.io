    Apa tujuan dibuatnya website Movie World ini?
    Tujuannya untuk menyelesaikan tugas akhir mata kuliah Interaksi Manusia dan Komputer dan mempelajari html, css dan jQuery

    Bagaimana cara membuat website Movie World ini?
    Pertama mencari referensi design, lalu menentukan konten apa saja yang mau ada di website.
    Setelah itu masuk ke tahap sketsa, setelah di setujui baru mulai membuat wireframe dan visual site map.
    Setelah semua tata letak setiap komponen bisa diketahui melalui wireframe, baru mulai menyusun mockup dan design.
    Sambil membuat mockup, anggota kelompok mulai mencari2 konten untuk websitenya. Anggota lainnya mulai mencari2 toturial di Youtube dan w3schools.
    Programmer mulai melakukan coding di Notepad++ dengan referensi dari css-tricks dan w3schools.
    Setelah semua coding selesai, ada anggota kelompok yang bertugas meng-hosting website melalui github.
    Terakhir, website yang sudah di hosting mulai bisa di uji coba dengan beberapa browser di perangkat PC, Mobile dan Tablet.
    Tahap akhir adalah finalisasi dan maintance website untuk mengurangi kemungkinan bug.

    Apa saja komponen untuk membuat website ini?
    Untuk komponen lengkapnya bisa di lihat di tab About Us

    Apakah website ini responsif?
    Setelah di uji coba, website ini responsif di perangkat mobile maupun PC.
    Karena css-nya menggunakan layout 12 grid sedangkan ukuran komponennya menggunakan % (bukan px).
    Sehingga otomatis setiap komponen akan me-resize dengan ukuran layar browser.
